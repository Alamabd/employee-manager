## Installation 📦

Automatic
```bash
git clone https://github.com/Alamabd/employee-manager
./install.sh
```

manual
```shell
git clone https://github.com/Alamabd/employee-manager
cd client
npm i
cd ..
cd server
npm i
cd ..
```
